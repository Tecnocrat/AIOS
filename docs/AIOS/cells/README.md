# Cells Documentation

This directory contains per-cell documentation and templates for archiving cell metadata, integration notes, and discovery information.

Files:
- `aios-cell-alpha.md` — documentation for the `alpha` cell (verified endpoints and probes).
- `cell_template.md` — template to record per-cell metadata after birth.

Use the `scripts/emit-waypoint.ps1` script to emit waypoints when you complete verification steps.
