## Summary

This document provides an introduction to Azure Management and Governance, focusing on cost management. It outlines the shift from CapEx to OpEx, key factors influencing costs in Azure, and introduces tools like the Pricing Calculator and TCO Calculator. It emphasizes the importance of monitoring, deprovisioning, and understanding the pay-as-you-go model.

## Key Concepts

*   **Capital Expenditure (CapEx):** Upfront investment in physical infrastructure.
*   **Operational Expenditure (OpEx):** Ongoing expenses for running and maintaining infrastructure.
*   **Pay-as-you-go Model:** Azure's billing model where users only pay for the resources they consume.
*   **Pricing Calculator:** A tool to estimate the cost of Azure resources.
*   **Total Cost of Ownership (TCO) Calculator:** A tool to compare the cost of running on-premises infrastructure with Azure.
*   **Microsoft Cost Management Tool:** A tool for monitoring and controlling Azure costs.
*   **Tags:** Metadata labels applied to Azure resources for organization and cost tracking.
*   **Resource Type:** Different Azure services and instances have varying costs.
*   **Consumption:** The amount of resources consumed directly impacts costs.
*   **Maintenance:** Cost associated with resource monitoring and optimization.
*   **Geography:** The location of resources affects costs due to regional differences.
*   **Subscription Type:** Different subscription plans provide varying usage allowances.

## Technical Details

*   **Resource Type Specifics:** Cost varies based on specifics of resources such as storage account type (blob, performance tier) and Virtual Machine characteristics (licensing, CPU cores).
*   **Consumption-Based Billing:** Azure charges based on actual usage of resources, allowing for flexibility and potential cost savings.
*   **Reserved Instances:** Users can reserve Azure resources for discounts of up to 72%.
*   **Cost Factors:** Costs are influenced by factors such as resource type, settings, region, network traffic, and subscription type.
*   **Practical Cost Control:** Emphasizes the importance of resource monitoring, and deprovisioning unused resources to manage costs.
